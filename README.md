# ArcticTernTech Homebrew Tap

ArcticTernTech 内部工具的 Homebrew 分发仓库。

---

## 使用方式

```bash
brew tap ArcticTernTech/tools
brew install harris-cli
```

## 升级

```bash
brew upgrade harris-cli
```

---

## 可用工具

| 工具 | 说明 |
|---|---|
| `harris-cli` | 跨境电商运营 CLI，支持 Amazon 多账号管理 |

---

## 说明

此仓库的 Formula 文件由 [harris-cli](https://github.com/ArcticTernTech/harris-cli) 的 GitHub Actions 自动更新，无需手动维护。
